How To Play Sudoku Master

Start a Game

To start a new game select “New Game” in the start screen.
Then select options and hit “Play”

Options

  • Difficulty: Easy, Medium, Hard, Very Hard.
    – Warning: Very Hard puzzels might not be uniquely solvable!
  • Timer: Timer can count down or up.
  • Background: Different background images are availible.

Options can be set in the start screen and from the menu during a game.

Gameplay

To fill in a number in the Sudoku board select an empty square with the mouse.
Then hit the number to be filled in in the keyboard.
Invalid entries will be highlighted in red.
At any point during the game select Help->Check Progress to check with entries are correct.
Press the pause button to pause the timer, and the resume button to resume.

Save and Load Games

To save the current gamestate select File->Save from the menu.
Games will be saved in the “saves/” directory.
To load a previously saved game select “Load Game” from the start screen or File->Load from the menu in the main screen.
